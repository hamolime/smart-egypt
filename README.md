# Smart Egypt 🇪🇬

AI-powered Egypt tourism web app built with React, Vite, and Express.

## Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Express 5 + Groq AI
- Languages: 7 languages including Arabic RTL

## Features
- AI Chatbot (Groq llama-3.3-70b)
- AI Trip Plan Generator
- Explore 12 Egyptian destinations
- Dark/Light theme
